HTML

  • The head element contains information about the webpage.
  • The meta elements contain information about the page that is used by the browser.
  • The body element represents the visible content shown to the user.
  • The title element does not show up anywhere except the tab/title bar and can improve search engine rankings.
  • The header element appears at the top of the page and usually contains a navigation menu and/or introductory content.
  • The footer element appears at the bottom of the page and usually contains the author, copyright, contact, sitemap, and navigation.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • A selector defines the element or attributes to which the rules, or declarations, will apply.
  • Declarations contain two important components: the CSS property we want to apply and the value of the property.
  • Review the CSS box model.
  • Class attributes allow us to share a CSS rule to any element we choose by assigning the rule to a class attribute with a class selector (i.e. card).

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git checkout main: switches to the main branch
  • git pull origin main: pulls down the latest main branch
  • git push origin feature/branch-name: pushes up all changes
  • git add -A: adds the changed files in the working directory to the staging area
  • git commit -m "message": commits changes with added message

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • We can use Chrome DevTools to view console.log outputs.
  • A value is considered 'falsy' if it is one of the following values: '0', '-0', 'On', '""', 'null', 'undefined', 'NaN', or a Boolean data type with the value of 'false'. All other values are considered 'truthy'.
  • An array is a single variable that is used to hold a group of data. They are typically used to hold data that is related in some way.
  • A 'for' loop uses the predictable pattern of indices to perform a task on all the items in an array by allowing a single block of code to be executed over and over.
  • A 'for' loop contains three important statements: for('starting point of loop', 'condition', 'what allows the array to iterate over each item').
  • A function is a set of instructions that tells the computer how to perform a certain task.
  • Declaring a function and executing a function are different; declaring a function creates a reusable block of code that performs a specific task, while executing a function actually runs the function in the spot we want it to.
